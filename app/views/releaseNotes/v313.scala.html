ScalaTest/Scalactic 3.1.3 Release Notes

ScalaTest/Scalactic 3.1.3 (for Scala 2.10, 2.11, 2.12, and 2.13; on the JVM, JavaScript, native, and Dotty) includes the enhancements and bug fixes listed below. No source code using ScalaTest/Scalactic 3.1.0 should break, and this release is binary compatible with 3.1.2.

For information on how to include ScalaTest in your project, see the install page. For information on how to use Scalactic in your production code, see its install page.


ScalaTest

ScalaTest 3.1.3 is a minor release of ScalaTest that makes a few changes, and supports Scala.js 1.1.1 and Dotty 0.26.

Bug Fixes

  • Fix IllegalAccessException when comparing 2 instances of class has private fields. (PR#1859)
  • Only expect non-filtered Async tests to be reported as completed in AsyncEngine, ParallelTestExecution and RandomTestOrder, which causes incorrect test count reported. (PR#1838)
  • In SuiteSortingReporter, only remove suite reporter after the SuiteCompleted event is reported correctly. (PR#1850)

Improvements

  • Added support of asynchronous test in scalatest native build. (PR#1867)
  • Improved Futures support by avoiding polling in Scala futures. (PR#1794)

Deprecations

ScalaTest 3.1.3 contains no new deprecations.


Scalactic

Improvements

  • Updates to support Scala.js 1.1.1.
  • Updates to support Dotty 0.26.

Deprecations

Scalactic 3.1.3 contains no new deprecations.
