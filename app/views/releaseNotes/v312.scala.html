ScalaTest/Scalactic 3.1.2 Release Notes

ScalaTest/Scalactic 3.1.2 (for Scala 2.10, 2.11, 2.12, and 2.13; on the JVM, JavaScript, native, and Dotty) includes the enhancements and bug fixes listed below. No source code using ScalaTest/Scalactic 3.1.0 should break, and this release is binary compatible with 3.1.2.

For information on how to include ScalaTest in your project, see the install page. For information on how to use Scalactic in your production code, see its install page.


ScalaTest

ScalaTest 3.1.2 is a minor release of ScalaTest that makes a few changes, and supports Scala.js 1.0.1 and Dotty 0.23.

Improvements

  • Updates to support Scala.js 1.0.1.
  • Updates to support Dotty 0.23.

Deprecations

ScalaTest 3.1.2 contains no new deprecations.


Scalactic

Improvements

  • Updates to support Scala.js 1.0.1.
  • Updates to support Dotty 0.23.

Deprecations

Scalactic 3.1.2 contains no new deprecations.
