ScalaTest/Scalactic 2.2.5 Release Notes

ScalaTest/Scalactic 2.2.5 includes just one change from 2.2.4, a bug fix for the DiagrammedAssertions macro that was accidentally left out of 2.2.4. No source code using ScalaTest/ScalaUtils 2.0 or higher should break, but from any version other than other 2.2.x releases (with which this release is binary compatible), you will likely need to do a clean build to upgrade.

For information on how to include ScalaTest in your project, see the download page. For information on how to use Scalactic in your production code, see its download page.

Bug fix

  • Fixed a problem encountered when DiagrammedAssertions was used with new. (reported as github issue #577)

Visit ScalaTest Release Notes for links to the release notes of all previous versions, or step back in time by visiting the release notes for the previous version.
