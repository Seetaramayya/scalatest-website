ScalaTest/Scalactic 3.2.4 Release Notes

ScalaTest/Scalactic 3.2.4 (for Scala 2.10, 2.11, 2.12, 2.13, and 3.0.0-RC1; on the JVM, JavaScript, and native) includes the enhancements and bug fixes listed below.

Although this release is binary compatible with 3.2.3, we unfortunately let one source-breaking change related to futureValue needing empty parens slip in, so we recommend you upgrade to 3.2.5 instead.

For information on how to include ScalaTest in your project, see the install page. For information on how to use Scalactic in your production code, see its install page.


ScalaTest

ScalaTest 3.2.4 is a minor release of ScalaTest that makes a few changes, and supports Scala.js 1.3.0 and Scala 3.0.0-RC1.

Bug Fixes

ScalaTest 3.2.4 contains no bug fixes.

Improvements

  • Improved support for Scala 3.
  • Upgraded to Scala 3.0.0-RC1.
  • Upgraded to use Scala 2.13.4
  • Upgraded to use Scala Native 0.4.0
  • Changed to use String's .linesIterator in NumericString.

Deprecations

ScalaTest 3.2.4 contains no new deprecations.


Scalactic

Improvements

  • Updated Dotty support Scala 3.0.0-RC1.

Deprecations

Scalactic 3.2.4 contains no new deprecations.
